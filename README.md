# AbstractAnalysis.jl

*Countable magma and semigroup for abstract functional analysis*

[![DOI](https://zenodo.org/badge/1291456483.svg)](https://doi.org/10.5281/zenodo.21394212)
[![Build status](https://ci.appveyor.com/api/projects/status/33kcuucnfrdhr4ov?svg=true)](https://ci.appveyor.com/project/chakravala/abstractanalysis-jl)

Implementation of `Semimagma` objects to enable the study of magmas, semigroups, groups, and more.
It also includes formalisms for a `CountableArray` and `SequenceArray` to facilitate the study of inductive or recursive `Semimagma` objects and also a fast `FunctionArray` kernel.
Functional analysis with `Limit` methods is enabled with magma algebra operations facilitated by embedded discrete dynamical systems in abstract metric spaces, typically in Banach spaces or induced from Hilbert spaces.
