# AbstractAnalysis.jl

*Countable magma and semigroup for abstract functional analysis*

Implementation of `Semimagma` objects to enable the study of magmas, semigroups, groups, and more.
It also includes formalisms for a `CountableArray` and `CountableCache` to facilitate the study of inductive or recursive `Semimagma` objects and also a fast `FunctionArray` kernel.
Further analysis with `FixedPoint` methods is facilitated from abstract magma operations.
