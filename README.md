# CountableMagma.jl

*Abstract magma and semigroup toolkit*

Implementation of `Semimagma` objects to enable the study of magmas, semigroups, groups, and more.
It also includes formalisms for a `CountableArray` and `CountableCache` to facilitate the study of inductive or recursive `Semimagma` objects.

This package is not really inspired from the Magma computer algebra system, it is based on the specification and requirements associated with the studies of the author.
It's not intended to be a large scale computer algebra implementation, rather it is a toolkit based on the usage and requirements of the author.

Undocumented features will be explained once the design matures.
