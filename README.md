# AbstractAnalysis.jl

*Countable magma and semigroup for abstract functional analysis*

Implementation of `Semimagma` objects to enable the study of magmas, semigroups, groups, and more.
It also includes formalisms for a `CountableArray` and `SequenceArray` to facilitate the study of inductive or recursive `Semimagma` objects and also a fast `FunctionArray` kernel.
Functional analysis with `Limit` methods are enabled with magma algebra operations facilitated by embedded discrete dynamical systems in abstract metric spaces, typically in Banach spaces or induced from Hilbert spaces.
